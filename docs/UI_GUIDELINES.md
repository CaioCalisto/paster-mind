# UI Guidelines

Style:
- minimal
- modern
- native macOS

Inspired by:
- Raycast
- Spotlight
- CleanShot X

Rules:
- avoid crowded layouts
- use generous spacing
- avoid excessive borders
- prioritize keyboard navigation
- support dark mode
- smooth animations

Window behavior:
- floating panel
- opens centered
- ESC closes
- CMD+SHIFT+V toggles visibility

Typography:
- use native fonts
- prefer medium weights
- avoid tiny text
